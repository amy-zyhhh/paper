# Seismic damage prediction in concrete gravity dams using a new framework based on Machine learning

## 摘要

<p>In this study, a comprehensive machine learning-based framework is presented for predicting seismic damage levels in concrete gravity dams in five different classes. The main objective of this study was to evaluate the impact of Feature Engineering approach compared to the traditional Feature Selection method on the performance of classification and ensemble learning models. For this purpose, optimal intensity measures (IMs) are used as input features, and a Global Damage Index ( D I G ), defined based on area-weighted local damage indices, is used as the output variable. The damage index data are obtained through nonlinear finite element analysis using Abaqus software. The main innovation in this research is the generation of a new composite feature from the intelligent combination of three optimal features (ASI, Tm and SED) that enhances the ability of the models to extract hidden patterns. The results show that the use of the new composite feature has significantly improved the performance of all the models studied in key evaluation criteria such as Accuracy, Precision, Recall, Specificity, F1-Score and AUC. This improvement was especially significant for models such as KNN and XGBoost, which had poorer performance in the feature selection approach. Finally, the feature engineering approach resulted in stable and reliable performance in all models and enabled them to best classify intermediate damage classes. This framework can be used as an efficient tool for pre-earthquake crisis management and prediction of overt and even covert damage in infrastructure structures.</p>

<h2>AI 分析</h2>
<p><strong>文章做了什么：</strong>文章提出了一种基于机器学习的综合框架，用于将混凝土重力坝的地震损伤水平预测为五个不同等级。核心工作是比较特征工程（Feature Engineering）与传统特征选择（Feature Selection）方法对多种分类和集成学习模型性能的影响。</p>
<p><strong>为什么做：</strong>旨在解决传统特征选择方法可能在高维地震动参数中提取有效信息不足的问题，特别是提升模型对中间损伤等级的识别能力，为震前危机管理和基础设施损伤（包括隐蔽损伤）预测提供更高效的工具。</p>
<p><strong>怎么做：</strong>1. 数据生成：以Koyna坝为对象，在Abaqus中建立坝-库水-地基系统的有限元模型，采用混凝土损伤塑性（Concrete Damaged Plasticity, CDP）模型进行非线性时程分析，计算基于面积加权局部损伤指数的全局损伤指数（D_I_G）作为输出标签。
2. 特征处理：从20个地震动强度指标（IMs）中，通过特征选择方法确定了三个最优特征：ASI（加速度谱强度）、Tm（平均周期）和SED（比能量密度）。进而，通过智能组合这三个特征生成了一个新的复合特征（composite feature）。
3. 模型训练与评估：使用KNN、SVM、逻辑回归、决策树、XGBoost、LightGBM、Stacking、朴素贝叶斯和硬投票（Hard Voting）共9种模型。采用交叉验证和超参数调优，以准确率、精确率、召回率、特异性、F1分数和AUC等指标评估特征选择与特征工程两种策略下的性能。</p>
<p><strong>做得怎么样：</strong>根据提供的性能对比表（Table 7 &amp; 8），特征工程方法显著提升了大多数模型的各项评估指标。例如，KNN和XGBoost的准确率分别提升了约10.6%和6.1%，F1分数提升超过13%。Stacking和Hard Voting等集成模型也获得稳定提升。然而，逻辑回归模型在特征工程下性能有所下降。总体而言，特征工程使模型在所有损伤等级（尤其是中间等级）的分类上表现更稳定可靠。</p>
<h3>主要结论</h3>
<ul><li>特征工程方法在预测混凝土重力坝地震损伤等级方面，整体上优于传统的特征选择方法。</li>
<li>通过智能组合ASI、Tm和SED三个最优特征生成的新复合特征，能有效增强机器学习模型提取数据隐藏模式的能力。</li>
<li>所提出的机器学习框架可作为震前危机管理和基础设施显性/隐性损伤预测的有效工具。</li></ul>
<h3>亮点和创新点</h3>
<ul><li>创新性地提出并验证了针对地震动参数的特征工程策略（生成复合特征），超越了传统的特征选择。</li>
<li>系统性地比较了九种主流机器学习及集成模型在坝体地震损伤分类问题上的性能，并进行了详尽的超参数调优。</li>
<li>建立了从高保真非线性有限元分析（CDP模型）到机器学习分类的完整工作流程，用于生成和利用损伤数据。</li></ul>
<h3>局限性</h3>
<ul><li>研究基于单一的Koyna坝数值模型，结论的普适性有待在不同坝型、几何尺寸和材料参数下验证。</li>
<li>训练数据完全来源于数值模拟，未包含实际震害数据，模型的真实外推能力存在不确定性。</li>
<li>原文信息不足：未明确讨论模型的计算效率（训练/预测时间），这在工程实时应用中可能很重要。</li></ul>
<h3>可能的发展方向</h3>
<ul><li>将框架扩展到其他类型的水坝（如拱坝）或更广泛的基础设施地震损伤评估。</li>
<li>融合实际震害观测数据或物理实验数据，提升数据集的多样性和模型的泛化能力。</li>
<li>探索更复杂的特征工程方法（如基于深度学习的特征自动提取）以及可解释性AI（XAI）技术。</li></ul>
<h3>注意事项</h3>
<ul><li>该框架的性能严重依赖于底层有限元模型的准确性（如CDP模型参数、边界条件、地震动输入）。使用前需对数值模型进行充分验证。</li>
<li>模型在训练数据范围之外的极端地震动或损伤状态下（如D_I_G接近1）的预测性能未知，需谨慎使用。</li>
<li>文中逻辑回归模型在特征工程下性能下降，提示特征构造需与模型特性相匹配，不能盲目应用。</li></ul>
